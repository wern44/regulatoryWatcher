CSSF catalog reconciliation

Run the full 2×7 (entity × publication-type) filter-matrix crawl against the live CSSF regulatory-framework listing. Unlike the incremental “Discover from CSSF” button on the Catalog page, this mode walks every page of every cell and feeds the auto-retire sweep — regulations no longer visible under AIFM or Chapter 15 ManCo filters will be marked REPEALED.

Takes about a minute. Progress shows live on the run page. Auto-retire is gated on a successful run and a minimum row count, so partial crawls won’t accidentally wipe the catalog. Regulations added via regwatch keep-active are exempt from retirement.

See Discovery runs for the full history, per-cell breakdown, and retired counts.
